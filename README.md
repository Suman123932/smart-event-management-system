# Smart Event Management System

## Description
Smart Event Management System is a web-based application that helps users create, manage, and organize events efficiently.

## Tools Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How to Run
1. Install XAMPP
2. Copy project folder to htdocs
3. Import database in phpMyAdmin
4. Run project on localhost
